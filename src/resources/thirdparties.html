Used Projects and Third Party Libraries in Cantara

The open source project Cantara would not exist without a list of amazing other software projects and libraries. This file provides a list of all projects and libraries which have been used to build Cantara or which are used as libraries or packages in Cantara itself.

Built with Lazarus, FPC and GIT

Cantara has been built with the Lazarus IDE and the Free Pascal Compiler FPC. See licening in the FPC wiki for detailed license information.

GIT has been used or version control.

Used Lazarus/Pascal packages and libraries

Cantara uses the LCL released under a modified LGPL. See licening in the FPC wiki for detailed license information. Beside the standard libraries of Lazarus, the following packages have been used:

  • bgrabitmap: released under a modified LGPL
  • MetaDarkStyle: released under LGPL 2.1 or later

Other used libaries inside the project

  • pptxgenjs: released under MIT
